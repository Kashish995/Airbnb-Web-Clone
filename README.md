# 🏠 Airbnb Web Clone

A full-stack Airbnb-inspired web application that allows users to create, view, update, and delete property listings. The project demonstrates CRUD operations, server-side rendering, database integration, and responsive web design using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript.

---

## 🚀 Features

* Create new property listings
* View all available listings
* View detailed information for each property
* Update existing listings
* Delete listings
* Responsive user interface
* MongoDB database integration
* RESTful routing architecture

## ⚡ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Airbnb-Web-Clone.git
cd Airbnb-Web-Clone
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the root directory:

```env
MONGO_URI=your_mongodb_connection_string
PORT=3000
```

### Run the Application

```bash
node app.js
```

or

```bash
npm start
```

Visit:

```text
http://localhost:3000
```
